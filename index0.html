This is migration 128a. This one was part 1.

This migration was and will continue to be hectic. Not only did it trigger hours before the Go Fest Challenge Week 1 event properly finished all around the globe, but it also had Week 2 right around the corner. Frameshift migration is the name of the game here!

So we settled in the following plan of action: today, migration day, we have conducted a premature research and report labeled as migration 128a. These nests will only be valid as long as the event isn't live in the nest's timezone, so gradually they will be changed until the whole world is live with the event. At that point, we'll proceed with migration 128b, so we have an accurate report until the event is finished.

We will probably also have a migration 128c once the event wraps up midway through this migration cycle so we'll see about that then. As always, feel free to report any and all errors in this migration's report over our Discord server [here].


Other than that, all the same. As usual, some species are absent from major nests and that's just bad luck. We'll try to add more nests if we find any that are actually good but that's always a maybe. Again thanks for stopping by; stay healthy, stay inside and happy hunting in these uncertain times!
