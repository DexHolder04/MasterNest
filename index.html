This is migration 138. This one was a time bomb.

I am honestly surprised Niantic managed to go a whole migration cycle without triggering any framshift migrations. Good for us, as it makes it easier to provide you all with an accurate report that will last for a decent amount of time.

However! This report has an expiration date with the upcoming Kalos Special Event which starts on December 02 at 10:00 AM local time. Once Gen 6 Pokémon get added to the nest pool it will make a mess of things (and as we're sitting at exactly 100 nesting species, maybe we'll even see some species be dropped from the pool to keep that round number). It will be chaos but we're here for it, and we'll definitely update the report once the event has started worldwide and we can assert an accurate report.

So look forward to that! In the meantime here's our report, we hope you find it useful until the fated date; if you notice any inaccuracies please report them to any member of staff over in our Discord server [here]. We're only human, mistakes are bound to happen!


Other than that, all the same. As usual, some species are absent from major nests and that's just bad luck. We'll try to add more nests if we find any that are actually good but that's always a maybe. Again thanks for stopping by; stay healthy, stay inside and happy hunting in these uncertain times!
